# CandlePilot 开发状态

最后更新：2026-07-14

本文件是项目实现状态的唯一进度台账。每完成一个功能，必须在同一个 Git 提交中更新本文件。

## 当前结论

CandlePilot 已具备本地 MVP 的主要骨架：订阅认证 LLM、币安公共行情、动态选币、
确定性风控、模拟成交、测试网基础下单、审计、历史回测和中文控制台。

当前版本仍不适合真钱实盘，也未达到长期无人值守运行标准。近期主线是实时行情、
完整模拟撮合、测试网对账与恢复。

## 更新规则

1. 每个可独立验收的功能使用一个 Git 提交。
2. 功能代码、测试、文档和本文件必须放在同一个提交中。
3. 只有测试通过并完成必要的实际检查，才能将项目标记为 `[x]`。
4. 部分实现使用 `[~]`，尚未开始或不可用使用 `[ ]`。
5. 不以盈利作为完成标准，以无前视偏差、可审计、风险边界和故障恢复为验收标准。
