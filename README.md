# CandlePilot

CandlePilot is a local-first, auditable LLM trading system for Binance USDⓈ-M
perpetual futures. The initial release is intentionally limited to backtesting,
paper trading, and Binance testnet execution.

The implementation plan is documented in [PLAN.md](PLAN.md).

## Development status

The project is under active development. Real-money trading is not available.
